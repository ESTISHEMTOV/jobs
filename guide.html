🛠️ מדריך תחזוקה — לוח המשרות

איך לשנות את הלוח בעצמך — גם בלי Claude. הכול חי במאגר ה-GitHub שלך.

🔗 קישורים מהירים למאגר שלך

  • המאגר (GitHub): github.com/ESTISHEMTOV/jobs
  • הסקריפט (המוח): scripts/refresh.mjs  ·  עריכה ✏️
  • קובץ התזמון (שעה): .github/workflows/daily-jobs.yml
  • ריצות / הרצה ידנית: לשונית Actions
  • מפתח Gemini (Secret): Settings → Secrets → Actions
  • העמוד הציבורי: estishemtov.github.io/jobs

איך זה עובד

  • כל בוקר ב-07:00 (שעון ישראל) רץ אוטומטית GitHub Action.
  • הוא מריץ סקריפט שאוסף משרות ומבקש ממודל בינה (Gemini, חינמי) לסנן ולסדר אותן.
  • התוצאה נשמרת, ו-GitHub Pages מציג את העמוד. אין תלות בשום חשבון חיצוני.

⭐ הדרך הכי קלה לשנות (מומלץ)

  1. פותחים את scripts/refresh.mjs ב-GitHub ומעתיקים את כל התוכן.
  2. מדביקים לכל עוזר AI שיש לך (ChatGPT / Gemini / Claude / כל אחד) וכותבים מה רוצים, למשל:
    "זה סקריפט שאוסף משרות. תוסיף לי את האתר X כמקור נוסף, ותחזיר לי את הקובץ המלא."
  3. מעתיקים את הקובץ שחזר → ב-GitHub לוחצים על העיפרון ✏️ בקובץ scripts/refresh.mjs → מוחקים הכול → מדביקים את החדש → Commit changes.

להוסיף אתר מקור — ידנית

בראש scripts/refresh.mjs יש רשימה בשם BOARDS. כל שורה = ['שם', 'כתובת חיפוש']. מוסיפים שורה:

const BOARDS = [
  ['JobMaster', 'https://www.jobmaster.co.il/jobs/?q=מנהל מערכות מידע'],
  ['AllJobs',   'https://www.alljobs.co.il/SearchResultsGuest.aspx?freetxt=מנהל מערכות מידע'],
  // ↓ הוסיפי שורה כזו עם האתר החדש:
  ['שם-האתר',  'https://כתובת-החיפוש/?q=מנהל מערכות מידע'],
];

שומרים (Commit) — ובהרצה הבאה האתר ייכלל.

פעולות נפוצות

  • לשנות שעת ריצה: בקובץ .github/workflows/daily-jobs.yml, השורה cron: '0 4 * * *' היא ב-UTC (04:00 UTC = 07:00 בישראל). ל-08:00 בישראל → '0 5 * * *'.
  • להריץ עכשיו (ידנית): לשונית Actions → Daily Jobs Refresh → Run workflow.
  • להחליף מפתח Gemini: מפתח חדש ב-aistudio.google.com/apikey → ואז Settings → Secrets and variables → Actions → GEMINI_API_KEY → Update.
  • לבדוק אם הריצה הצליחה: לשונית Actions — ✓ ירוק = הצליח.
  • לשנות תפקידים מבוקשים / אזורים / סינונים: בדרך הקלה למעלה — בקשי מעוזר AI לערוך את הכללים ב-scripts/refresh.mjs.
טיפ: אחרי כל שינוי — לכי ל-Actions → Run workflow כדי לראות את התוצאה מיד (אחרת זה יתעדכן ממילא בבוקר הבא).

תקלות נפוצות

  • העמוד לא התעדכן: ב-Actions יש ריצה אדומה? לרוב זה מפתח Gemini שפג/חרג ממכסה — צרי מפתח חדש ועדכני את ה-Secret.
  • הדפדפן מציג גרסה ישנה: רענון קשיח — Ctrl+F5.
  • קישור משרה מוביל ל-0 תוצאות: נדיר; הקישורים הם חיפושים באתר המקור — חפשי שם לפי שם החברה.
